HTML

  • The head element contains information about the webpage.
  • Head elements are not visible to the user of a website.
  • The body element represents the visible content shown to the user.
  • The user can only interact with code placed within the body element.

CSS

  • Margins indicate the amount of space around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class selectors are designated/preceeded by a full stop.
  • You can style HTML elements using the elements as selectors or by using the class attribute as a selector.

CSS formats

Inline

  • Apply to a specific HTML tag.
  • Quick and permanent changes.
  • Must edit individually.

Internal

  • Embeded directly within the HTML file (head element), using a style element.
  • Easily edit all element within a page, with changes applying to all tagged elements.

External

  • Uses an entirly separate file for CSS rules.
  • When using an extenal CSS file, use a link element placed in the head element of the HTML file.

GIT

  • Git cheatsheet available via computer bookmarks.
  • Remember to look at what branch you are on before commencing work!
  • Navigate branches via Gitbash.
  • Git checkout -b (new branch).
  • Git add -A (stage all existing changes in working branch to commit).
  • Always ensure that current branch is up to date before opening a pull request (git pull origin main).

Summarise the Git flow

  • Stage changes.
  • Add commit message.
  • Ensure local branch is up to date (pull base branch into feature branch).
  • Push to remote repo.

Javascript

  • A variable is a named container for the purpose of storing data in our code.
  • Variables must first be defined in order to have any meaning.
  • Control flow is the order in which a computer executes code in a script. In JavaScript, from top to bottom.
  • Controll flow can be interupted, altered or ignored using conditional statements and loops.
  • A function is a block of code that performs a specific task.
  • Unilke a for loop, functions must be called upon to execute, and will not execute automatically.
